Un generatore interattivo di prompt per Sora 2, il modello di generazione video AI di OpenAI. Questo tool aiuta gli utenti a creare prompt dettagliati e ben strutturati per ottenere i migliori risultati da Sora 2.

## ğŸš€ Come Usare

1. **Download**: Scarica o clona questa repository
2. **Apertura**: Apri il file `pages/index.html` nel tuo browser web preferito
3. **Compilazione**: Compila i campi del form con le caratteristiche desiderate per il tuo video:
   - **Soggetto**: Descrivi cosa vuoi vedere nel video
   - **Stile**: Specifica lo stile visivo (cinematografico, anime, realistico, etc.)
   - **Movimento**: Definisci il tipo di movimento della camera o dei soggetti
   - **Ambientazione**: Descrivi dove si svolge la scena
   - **Illuminazione**: Specifica le condizioni di luce
   - **Durata**: Indica la lunghezza desiderata del video
   - **Dettagli Aggiuntivi**: Aggiungi qualsiasi altra specifica
4. **Generazione**: Clicca sul pulsante per generare il prompt
5. **Copia**: Usa il pulsante di copia per copiare il prompt e usarlo con Sora 2
